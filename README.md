# 說明
可以依照裝置螢幕切換表格模式

這個表格會依照螢幕解析度，自動切換不同表格模式
- 桌機模式：桌機模式的欄位標題於上方，資料於下方條列
- 表格模式：行動模式的欄位標題於左方，資料於右方條列

判斷螢幕寬度預設為 768 px

# 安裝
```
npm install response-table
```

# 使用方式
```
<ResponseTable :data="tableData" :label="" style="width: 100%" />
```
